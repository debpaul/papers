\hypertarget{generate-tex}{%
\subsubsection{Generate Tex}\label{generate-tex}}

The \texttt{-C} or \texttt{-\/-citeproc} flag generates citations using
\texttt{cls} and \texttt{bibliography} directives from a markdown file.

\begin{Shaded}
\begin{Highlighting}[]
\ExtensionTok{pandoc} \AttributeTok{{-}C} \AttributeTok{{-}s}\NormalTok{ test.md }\AttributeTok{{-}o}\NormalTok{ test.tex}
\end{Highlighting}
\end{Shaded}

\hypertarget{generate-pdf}{%
\subsubsection{Generate PDF}\label{generate-pdf}}

\begin{Shaded}
\begin{Highlighting}[]

\end{Highlighting}
\end{Shaded}
